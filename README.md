## Installation

### Prerequisites
* `bower` is installed
* `npm` is installed
* `yo` (yeoman) is installed globally
* `generator-angular` is installed


### Building

run `bower install` in the directory

### Running / Starting the Project Server

run `grunt serve` in the directory
